Phsbot声明

  • 本机器人和游戏内容无关，请不要再反馈什么：
    • 我升级了Phsebot xx版本，用了xx插件，你看这个聊天信息变成什么样子
    • 你看这个老是出现什么乱七八糟的乱码(注：pfe信息颜色不是乱码！！！！)

Phsebot命令表

命令 介绍 用法
bindid 绑定xboxid <<bindid “$1”
start 开服 <<start
unbind 解绑xboxid <<unbind %qqid%
motd motd请求 <<motd xxx.xxx.xxx:19132（端口可省略）

FakePlayer命令

命令 介绍 用法
add 添加假人 <<FakePlayer add “tester” <皮肤:可选Steve或Alex> <是否允许聊天控制:可选>
remove 删除假人 <<FakePlayer remove “tester”
disconnect 断开假人连接 <<FakePlayer disconnect “tester”
connect 连接假人 <<FakePlayer connect “tester”
setchat 设置是否可以聊天控制 <<FakePlayer setchat “tester” true(该参数不填默认为false)
getstate 获取假人状态 <<FakePlayer getstate “tester”
list 获取所有假人 <<FakePlayer list
removeall 删除所有假人 <<FakePlayer removeall

Phsebot占位符 - QQ信息类

占位符 介绍 适用范围
%qqnick% qq昵称 群消息
%qqid% qq号 群消息
%xboxid% qq号绑定的xboxid（没绑定默认为%xboxid%） 群消息

Phsebot占位符 - 状态信息类

占位符 介绍 适用范围
%cpu% cpu占用率 群消息&控制台
%ram_1% 内存占用百分比 群消息&控制台
%ram_2% 剩余内存 群消息&控制台
%ram_3% 已用内存 群消息&控制台
%ram_4% 总内存 群消息&控制台
%server_motd% 服务器motd 群消息&控制台
%server_version% 服务器版本 群消息&控制台
%server_online% 在线玩家数 群消息&控制台
%server_maxonline% 最大玩家数 群消息&控制台
%server_levelname% 服务器存档名 群消息&控制台
\n 换行 群消息&控制台

Phsebot占位符 - 时间类

占位符 介绍 适用范围
%date% 当前日期（2021-09-13） 群消息&控制台
%time% 当前时间（21:23:11） 群消息&控制台
%year% 当前年份（2021） 群消息&控制台
%month% 当前月份（09) 群消息&控制台
%week_n% 当前星期（2）[1-7]。周一为1，以此类推。 群消息&控制台
%week_t% 当前星期（一）[日、一、二、三、四、五、六] 群消息&控制台
%week_j% 当前星期（月）[日、月、火、水、木、金、土] 群消息&控制台
%day% 当前天数（13) 群消息&控制台
%hour_12% 当前小时（9），十二小时制 群消息&控制台
%hour_24% 当前小时（21），二十四小时制 群消息&控制台
%ampm% 当前是上午（AM）还是下午（PM），答曰：PM 群消息&控制台
%min% 当前分钟（23) 群消息&控制台
%sec% 当前秒数（11) 群消息&控制台
